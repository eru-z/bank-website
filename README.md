# Bank Website (PHP & MySQL)

A banking web application built using core PHP and MySQL.

## Features
- User authentication (login & register)
- User dashboard
- Bank cards overview
- Money transfer page
- Loan management section
- Banking quizzes and lessons
- Session handling and logout
- Database connection using PHP

## Tech Stack
- PHP (core PHP)
- MySQL
- HTML
- CSS
- JavaScript

## Pages Overview
- index.php – landing page
- login.php / register.php – authentication
- dashboard.php – user dashboard
- cards.php – bank cards
- transfer.php – money transfer
- loan.php – loan section
- lessons.php – learning content
- quizes.php / quiz1.php / quiz2.php / quiz3.php – quizzes
- db.php – database connection

## Purpose
This project was built to practice backend development, authentication, session handling, and database-driven PHP applications.
